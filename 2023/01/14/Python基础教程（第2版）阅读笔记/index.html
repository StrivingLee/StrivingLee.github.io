一、基础知识

Python 的碎碎念

Python 起源于一个幽默团体，而非蟒蛇（

Python 的其他版本

ActivePython 内核与标准版相同，最大的区别是包含了许多额外可用的模块

Stackless Python 最大优点是允许深层次递归和更高效地执行多线程

运算符

双斜线 // 是 C 语言中除法

** 是幂次，注意幂次的优先级比取反要高，所以 $(-3)^2$ 应写为 (-3) ** 2

长整型数

长整型数和普通整数可以混合使用，除非需要进行类型检查（极少）

十六进制和八进制数

0xAF、010

获取用户输入

input 会假设用户输入的是合法的 Python 表达式（和 repr 相反）

raw_input 会把所有的输入当作原始数据放入字符串中

建议尽可能使用 raw_input

函数

内建函数

模块

需要使用 import 语句导入模块

注：int() float() long() 并非完全等同于函数，事实上他们是类型对象

这样就不用写模块的名字来调用函数了

在 python 中可以使用变量来引用函数（或者 Python 中大多数的对象），比如，通过 foo = math.sqrt 进行赋值，则 foo(4) 的结果为2.0

cmath 和复数

注意这里没有使用 from...import ，否则会引起逆天的命名冲突，就无法再使用普通的 sqrt() 函数了

Python 本身提供对复数的支持

Python 中没有单独的虚数类型，他们被看作实数部分为 0 的复数

通过命令提示符运行 Python 脚本

C:\>python hello.py

字符串

单引号和双引号

转义符号

使用长字符串和原始字符串（可以联合使用）减少绝大多数反斜线的使用

拼接字符串：+

值转换为字符串的两种机制：str 和 repr

str 函数会把值转换为合理形式的字符串，repr 会创建一个字符串，以合法的 Python 表达式的形式来表示值

长字符串

用三个单引号或者双引号引用跨行字符串（不需要反斜线进行转义）

原始字符串

在书写正则表达式的时候特别有用

以 r 开头，不能以反斜线结尾

Unicode 字符串

在 Python 3.0中所有字符串都是 Unicode 字符串

二、列表和元组

Python 通过负下标表示倒数第几个

序列概览

Python 6 种内建序列：列表、元组（最常用两种）、字符串、Unicode 字符串、buffer 对象、xrange 对象

列表可以修改，元组不能

一般都可以使用列表，例外：字典的键不能修改，使用元组

容器：包含其他对象的任意对象

序列和映射（例如字典）是两类主要的容器，其他的类型还有集合（set）等

通用序列操作

索引（indexing）、分片（sliceing）、加（adding）、乘（multiplying）、成员资格、计算序列长度、找出最大或最小元素

负数下标表示倒数第几个

索引

分片

如果分片所得部分包括序列结尾的元素，只需要将最后的索引置空即可

同样，置空的方法也适用于开始的元素

复制整个序列只需将两个索引都置空

步长

步长不能为0，但是可以为负数，表示从右向左提取

序列相加

类似于字符串，序列可以通过 + 号进行拼接

注：只有相同类型的序列才能进行拼接

乘法

只占用空间，没有实际含义的元素——None

可以用于列表的初始化

成员资格

in 运算符，返回布尔值

列表——Python的“苦力”

用 list 由字符串创建列表

元素赋值

不能为位置不存在的元素赋值，和C是一样的

删除元素

del

分片赋值

在这里分片愈发显示出它的强大特性，甚至可以使用与原序列不等长的序列替换分片

列表方法

append ：追加对象

count ：统计某个元素出现的次数

extend ：用新序列扩展原有的列表

index

insert ：insert(subscript, content)

pop ：移除一个元素并且返回该元素的值，参数缺省则为最后一个

这里可以实现栈，至于队列的更好实现方式是，使用 collection 模块中的 deque 对象

remove ：移除第一个匹配项，改变列表但是不返回值

reverse ：反转列表，改变列表但是不返回值

sort

高级排序：**sort 方法默认升序排序**，如果想按自定义方法排序，那么可以自定义 compare(x, y) 函数，此函数在 x < y 时返回负数，x > y 时返回正数，相等时返回 0

还有另两个可选的参数——key 和 reverse（通过名字来指定的关键字参数）

元组——不可变序列

用逗号分割一些值便自动创建了元组

大部分时候是通过括号括起来的，空元组可以用没有内容的两个圆括号表示

一个值的元组：必须加括号，如 (42, ) ，很奇怪的语法，不是吗

tuple 函数：以一个序列作为参数病把它转化为元组

基本元组操作

仿照其他序列即可

元组的用武之地

在映射（和集合的成员）中当作键使用

三、字符串

字符串不可变！！！

所以分片赋值是不合法的

字符串格式化

字段宽度和精度

10.2f 字段宽10，精度2

用 * 作为字段宽度或者精度（或二者都用），此时从元组参数中读出数值

加上 + 表示无论正负都标识出符号

find 方法

返回最左端索引，否则返回 -1

类似于人工实现的查找方法，注意与 strstr 的区别和联系

可以只提供起始点也可以提供起终点

join、split 方法

互为逆方法，join 用于在队列中添加元素，split 通过指定分隔符分割字符串

join 需要的队列元素都是字符串

split 缺省分隔符是空格

大小写方法

lower、islower、upper、isupper、title、istitle、capwords、capitalize、swapcase

replace

translate

可以替换字符串中的某些部分，可以用于类似语言转换的功能，因此得名

四、字典：当索引不好用时

字典是 Python 中唯一内建的映射类型

值没有特殊的顺序

键可以是数字、字符串甚至是元组

建立字典

可以利用 dict 函数，通过其他映射或者键值对建立字典

和 list、tuple、str 一样，dict 也是类型而非真正的函数

字典和列表的重要区别

  • 键类型

  • 自动添加

    • 列表不能将值关联到不存在的索引上
    • 即使这个键原来不存在，字典也可以自动添加键值对

  • 成员资格

    • 表达式 k in d（字典） 查找的是键
    • 表达式 v in l（列表） 查找的是值

字典中的检查方式更高效，而且事实上两种方式是等效的

如果字典含有指定的键，那么相应的值也是唾手可得

字典方法

clear：清除字典中所有的项

原地方法，无返回值（返回 None）

copy：shallow copy

deepcopy：

from copy import deepcopy

fromkeys：根据给定键和默认值建立字典，默认值缺省为 None

get：访问存在的项时返回键对应的值，访问不存在的项时返回 None
